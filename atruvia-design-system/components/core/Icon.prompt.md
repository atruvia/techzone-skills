Official Atruvia icon from the sprite; use anywhere you need an icon.

```jsx
<Icon name="robot" size={24} />
<Icon name="arrow-right" title="Weiter" />
```

60 icons: arrows/carets, check, cross, plus, menu, mail, phone, smartphone,
database, ecosystem, robot, shaking-hands, hand-leaf, idea, location, pin,
calendar, time, download, edit, filter, play/pause, info/success/warning/error,
social (linkedin, instagram, facebook, twitter, xing, youtube), and more.
Tinted via `currentColor`. `sprite` prop overrides the path per page; pages deeper
than root pass e.g. `sprite="../../assets/icons/atruvia-icons.svg"`.
