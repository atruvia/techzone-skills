Pill-shaped action button; use for any primary or secondary action.

```jsx
<Button variant="primary" iconRight={arrowIcon}>Jetzt starten</Button>
```

Variants: `primary` (blue), `secondary` (bordered/neutral), `ghost` (text),
`gradient` (VIA-Line gradient — reserve for hero CTAs). Sizes: `sm | md | lg`.
Props: `block`, `iconLeft`, `iconRight`, `disabled`, `as` (e.g. `"a"`).
